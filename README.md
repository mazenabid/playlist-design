A sleek music playlist interface built with vanilla JavaScript, HTML, and CSS. This project dynamically renders playlist data using JavaScript's `.map()` method and creates reusable music cards with interactive hover effects.

## Features

- Dynamic playlist rendering from JavaScript data
- Modern card-based layout
- Responsive flexbox design
- Album artwork display
- Interactive hover animations
- Image glow effects on card hover
- Clean dark-mode inspired UI

## What I Learned

This project reinforced the importance of separating data from presentation. Instead of manually creating each playlist card, I used JavaScript's `.map()` method to generate the entire interface from an array of playlist objects, resulting in cleaner and more scalable code.
